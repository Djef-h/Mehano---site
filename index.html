✦ 02 — Code

Our Robot Code

We program our robot using Python on LEGO Spike Prime. Here's a snippet from our autonomous run code:

mehano_run1.py
# Team Mehano — FLL 2025-26 Robot Code
# Mission: Autonomous Run 1

from spike import PrimeHub, MotorPair, ColorSensor
import runloop

hub = PrimeHub()
motors = MotorPair('A', 'B')
color = ColorSensor('C')

async def main():
    hub.speaker.beep(440, 200)
    # Drive forward 40 cm
    await motors.move(40, 'cm', steering=0)
    # Turn right 90 degrees
    await motors.move(90, 'degrees', steering=100)
    # Complete mission
    hub.light_matrix.show_image('HAPPY')

runloop.run(main())
